# Bolsines

Proyecto de la materia **Diseño de Sistemas** (DSI) - UTN FRC.

Modelado de un sistema de gestión de bolsines mediante diagramas UML.

## Estructura

| Carpeta | Descripción |
|---------|-------------|
| `casos_de_uso/` | Diagramas de casos de uso y secuencia (PlantUML) |
| `diagrama_clase/` | Diagramas de clase (PlantUML) |
| `maquina_estado/` | Diagramas de máquina de estados (PlantUML) |
| `documentacion/` | Documentación del proyecto (PDFs, ERS, consignas) |
| `out/` | Salidas generadas (PNG/SVG/PDF) — ignoradas por git |

## Requisitos

- [PlantUML](https://plantuml.com/) para editar y generar los diagramas.
- [PlantUML para VSCode](https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml) — permite previsualizar y exportar diagramas directamente desde el editor.
